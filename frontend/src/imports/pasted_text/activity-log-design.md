Design the "Activity Log" page for the EWAY LMS Admin Dashboard.

IMPORTANT:
Use the same design system as the Admin Dashboard:

* dark glassmorphism theme
* same sidebar layout
* same top header style
* same spacing and card style
* do not redesign the sidebar

This page allows administrators to monitor system activities.

PAGE HEADER

Top link:
← Back to Dashboard

Page Title:
Activity Log
