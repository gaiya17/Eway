Design the "Chatbot Management" page for the EWAY LMS Admin Dashboard.

IMPORTANT:
Use the same design system as the Admin Dashboard:
- dark glassmorphism theme
- same sidebar layout
- same top header style
- same rounded cards
- same spacing and typography
- do not redesign the sidebar

The page is for the Admin to manage the LMS AI chatbot.

--------------------------------------------------

PAGE HEADER

Top link:
← Back to Dashboard

Page Title:
Chatbot Management

Subtitle:
Control chatbot settings, manage FAQs, and monitor chatbot usage.

+ Add FAQ
